- ⚡ **[SymbolCore](https://github.com/RicKanjilal/Symbolcore)** custom programming language with three syntax modes and hybrid Python/C compilation
- ♟️ **[MonstrosityVX](https://github.com/RicKanjilal/MonstrosityVX)** chess engine with iterative deepening, NMP, LMR, transposition tables
